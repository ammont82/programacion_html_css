Uso de la Etiqueta <pre>

La etiqueta <pre> permite mostrar texto con formato predefinido:

        Ejemplo de texto con espacios y saltos de línea.

        Línea 1: Hola, este es un ejemplo.
        Línea 2:     Este texto tiene tabulaciones.
        Línea 3: Y mantiene los espacios.

Como puedes ver, el texto dentro de la etiqueta <pre> se presenta tal como se escribe en el código fuente.
