Sección de Texto en Múltiples Columnas

El diseño en columnas múltiples permite dividir texto largo en varias columnas de manera automática.

Puedes controlar el número de columnas usando la propiedad column-count.

Además, puedes ajustar el espacio entre columnas usando la propiedad column-gap.

Experimenta con diferentes valores para ver cómo cambian las columnas.
